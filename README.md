Kidney Disease Prediction API.

*This project is a Machine Learning API that predicts whether a patient has Chronic Kidney Disease (CKD) based on medical attributes.

*The model was trained using a Decision Tree classifier and deployed as a REST API using Flask. The application is hosted on Render and
runs in a production environment using Gunicorn.

Tech Stack
*Python
*Flask
*Scikit-learn
*Pandas and NumPy
*Joblib (model serialization)
*Gunicorn (production server)
*Render (cloud deployment)
